Introduction: My site is for the people to read latest news about video games and technology. I chose to use hierarchical structure.


State 3 things that have inspired you when creating your website:

  • I am very interested in technology and video games are my passion.

  • I like to read news about both every day from various other sites.

  • I wanted to see how I can make site like this.



State 3 ways in which your site is accessible:

  • Alternative text for images and videos, if for some reason they could not be seen.

  • Consistent navigation

  • Site itself uses bright colours. This is especially useful for the text to be more readable. Although if someone prefers dark colours - dark mode is also available for this site



State 3 ways in which you have considered usability in your site:

  • Site is responsive; it can be used on multiple devices

  • Clarity; design of the website is simple and consistent, without unnecessary distractions. Every section of information is labelled. Navigation is straightforward.

  • Relevancy; if a user is interested in technology news section, she/he can easily navigate to this particular section to find needed information in this field.



What aspect(s) of your work do you think worked well and why?

I think the design is a well-made and any group of users can quickly familiarise themselves with the site’s layout.

Site is also responsive, but this is mostly thanks to the Bootstrap classes – I just adjusted them to my needs and add multiple @media queries to better suit my website.

I was also struggling a bit with progress bar code but eventually manage to make it work

Some CSS effects such as separating bars on the news pages.



What aspect(s) of your work could be improved, and how might you do things differently another time?

Maybe add more subpages, add something like search bar. In the news pages I could add horizontal image scroll. In the future I will try make a website with different design and layout.



More clear focus indicators - website is keyboard-friendly but some elements are unclear or do not work fully as intended.

What resources were used when creating your work?

My site was made based on Bootstrap template: Vlava - Vlava Template

Because of that some pieces of code in my project may be similar to the template’s. Everything used from the template is described and changed in at least some aspects. What was changed and what stayed the same is written exactly in the .html, .css, and .js files. Some of the code used from other sources is explained here, in the report.html file.

I have used Bootstrap v4.6 mainly for the flexbox grid, as well as some effects that helped my site look how I wanted (image carousel).

For the JS library I used the jQuery and AOS (Animate On Scroll Library).

Fonts were imported from the https://fonts.google.com/.

IcoFont (up-arrow and dark-mode switch) and flaticon (social media) icons were used.

If I wanted to check some CSS, jQuery, Bootstrap or HTML properties, methods, etc. I went mostly with https://www.w3schools.com/ references. All colours were taken from the Google RGB colour picker.
